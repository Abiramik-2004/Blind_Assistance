🚀 ECHOWAY – AI-Powered Navigation for the Visually Impaired

ECHOWAY is an advanced AI-powered mobile application designed to empower visually impaired individuals with safe, independent navigation. By integrating real-time object detection, intelligent voice assistance, and GPS-based navigation, the application acts as a reliable digital companion for everyday travel.

The system focuses on accessibility, safety, and ease of use by enabling hands-free interaction and continuous environmental awareness. It bridges the gap between physical mobility challenges and modern intelligent technology.

🌟 Key Features
🎤 Voice Command Navigation
Users can interact with the application entirely through voice commands. Destinations can be set without manual input, making the system highly accessible and user-friendly.

🗺️ Real-Time Navigation
The application integrates with Google Maps to provide accurate, real-time directions. Users receive step-by-step voice instructions, ensuring they stay on the correct path.

👁️ AI-Based Object Detection
Using advanced computer vision techniques, the system detects obstacles such as:
People
Vehicles
Staircases
Poles and objects
This ensures users are constantly aware of their surroundings.

🔊 Intelligent Voice Guidance
The system provides immediate auditory feedback:
“Caution! Staircase ahead. Watch your steps.”
This helps users react quickly and avoid potential hazards.

📍 GPS Tracking & Route Optimization
ECHOWAY continuously tracks the user's location and dynamically adjusts routes if deviations occur, ensuring safe navigation at all times.

🔐 Secure Authentication System
Users can log in securely using:
Email & Password
Google Sign-In
All user data is managed through Firebase for reliability and security.

⚙️ How It Works
The application follows a structured workflow to ensure smooth operation:
🎤 User provides destination via voice input
🔄 Speech is converted into text using speech recognition
🗺️ Route is calculated using map services
📷 Camera captures real-time surroundings
🧠 AI model detects obstacles continuously
🔊 Voice system alerts user about obstacles and directions
🎯 User safely reaches the destination

🧠 Technology Stack
Category	Technology Used
-> AI / Machine Learning	YOLOv8
-> Computer Vision	OpenCV
-> Mobile Development	Android
-> Backend	Firebase (Authentication + Firestore)
-> Maps & Navigation	Google Maps API
-> Voice Processing	Speech-to-Text & Text-to-Speech

🎯 Objective
The primary goal of ECHOWAY is to enhance the quality of life for visually impaired individuals by:
->Providing safe and reliable navigation
->Enabling real-time obstacle awareness
->Supporting voice-based interaction
->Promoting independence in daily activities
